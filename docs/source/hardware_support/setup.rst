Konfigurasi Laptop Baru
=======================

Saat laptop pertama kali dinyalakan, laptop perlu diatur terlebih dahulu sebelum dapat digunakan untuk tugas sehari-hari. Pengaturan ini mencakup:

* Pengaturan Awal
* Konfigurasi Sistem Operasi Windows
* Konfigurasi Akun Email (untuk klaim free Office (jika ada))
* Konfigurasi Software
