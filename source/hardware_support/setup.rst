Konfigurasi Laptop Baru
=======================

Saat laptop pertama kali dinyalakan, laptop perlu diatur terlebih dahulu sebelum dapat digunakan untuk tugas sehari-hari. Pengaturan ini mencakup:

* Pengaturan Awal
* Konfigurasi Sistem Operasi Windows
* Konfigurasi Akun Email (untuk klaim free Office (jika ada))
* Konfigurasi Software

-----

Pengaturan Awal
---------------





Instalasi Program Esensial
--------------------------

Berikut adalah katalog program esensial untuk berbagai kebutuhan.

Penerjemahan
^^^^^^^^^^^^

.. grid:: 2

    .. grid-item-card:: Paratext
      :link: https://paratext.org/download/

      Paratext adalah perangkat lunak utama untuk penerjemahan Alkitab yang digunakan oleh banyak tim penerjemah di seluruh dunia.
      +++
      Tertulis

    .. grid-item-card:: Render
      :link: https://github.com/faithcomesbyhearing/render/releases

      Render adalah perangkat lunak untuk penerjemahan Alkitab secara lisan (Oral Bible Translation/OBT), dikembangkan oleh Faith Comes By Hearing bersama mitra lainnya.
      +++
      Lisan

.. grid:: 2

    .. grid-item-card:: Audacity
      :link: https://audacityteam.org/download/

      Audacity adalah perangkat lunak gratis dan open source untuk merekam dan mengedit audio.
      +++
      Lisan

    .. grid-item-card:: One Story Editor
      :link: https://software.sil.org/onestoryeditor/download/

      One Story Editor adalah perangkat lunak untuk mendokumentasikan, mengorganisasi, dan melakukan pengecekan konsultan terhadap transkripsi cerita Alkitab secara lisan.
      +++
      Lisan

.. grid:: 2

    .. grid-item-card:: Bloom Editor
      :link: https://bloomlibrary.org/page/resources/all-bloom-installers

      Bloom Editor adalah perangkat lunak untuk membuat dan menerbitkan buku-buku sederhana, terutama untuk literasi dan pendidikan di komunitas bahasa minoritas.
      +++
      Tertulis, Lisan

Ortografi
^^^^^^^^^

.. grid:: 2

    .. grid-item-card:: WeSay
      :link: https://software.sil.org/wesay/download/

      WeSay adalah alat untuk pengembangan kamus komunitas yang memfasilitasi pengumpulan, pengelolaan, dan berbagi data leksikal untuk bahasa-bahasa yang sedang dikembangkan.

    .. grid-item-card:: Lexique Pro
      :link: https://software.sil.org/lexiquepro/download/

      Lexique Pro adalah perangkat lunak untuk menampilkan, mengelola, dan mendistribusikan data leksikal (kamus) yang telah dikumpulkan, misalnya dari FieldWorks atau WeSay.

.. grid:: 2

    .. grid-item-card:: FieldWorks Language Explorer (FLEx)
      :link: https://software.sil.org/fieldworks/download/

      FLEx adalah perangkat lunak linguistik untuk analisis dan pengelolaan data bahasa, termasuk pengembangan leksikon, analisis morfologi, sintaksis, dan pengelolaan teks.

Sumber Bahan/Referensi
^^^^^^^^^^^^^^^^^^^^^^

.. grid:: 2

    .. grid-item-card:: Logos
      :link: https://logos.com/get-started

      Logos adalah perangkat lunak studi Alkitab tingkat lanjut, digunakan oleh konsultan dan peneliti Alkitab.

    .. grid-item-card:: SABDA 4.30
      :link: https://sabda.net/

      SABDA adalah perangkat lunak Alkitab digital berbahasa Indonesia yang menyediakan berbagai versi Alkitab, alat pencarian, referensi silang, dan sumber-sumber pendukung studi Alkitab.

Productivity
^^^^^^^^^^^^

.. grid:: 2

    .. grid-item-card:: Google Chrome
      :link: https://google.com/chrome/

      Google Chrome adalah peramban web (browser) populer yang cepat dan mendukung berbagai ekstensi serta fitur keamanan terbaru.

    .. grid-item-card:: Foxit PDF Reader
      :link: https://foxit.com/downloads/#foxit-pdf-reader

      Foxit PDF Reader adalah aplikasi ringan untuk membaca dan mengelola dokumen PDF, menawarkan fitur anotasi, pengisian formulir, dan pencetakan.

    .. grid-item-card:: VLC
      :link: https://videolan.org/vlc/download-windows.html

    .. grid-item-card:: Zoom
      :link: https://zoom.us/download?os=win

      Zoom adalah aplikasi konferensi video yang banyak digunakan untuk rapat daring, webinar, dan kolaborasi jarak jauh.

.. grid:: 2

    .. grid-item-card:: LibreOffice
      :link: https://libreoffice.org/download/download-libreoffice/

      LibreOffice adalah paket aplikasi perkantoran gratis dan open source yang mencakup pengolah kata, spreadsheet, presentasi, dan lain-lain.

Typesetting
^^^^^^^^^^^
